    Flexbox es un modelo de diseño en CSS que organiza elementos en una fila o columna, ajustándolos automáticamente para que sean flexibles y responsivos.

  • Flexbox

    • Se usan para controlar el espaciado y diseño en una página web:

    Margin: Para separar elementos entre sí y evitar que se vean amontonados.

    Padding: Para dar espacio dentro de un elemento, evitando que el contenido toque los bordes.
