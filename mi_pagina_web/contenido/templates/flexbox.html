Flexbox

Flexbox es un modelo de diseño en CSS que se usa para crear layouts flexibles y adaptables en páginas web. Funciona asignando a un contenedor el estilo display: flex;, lo que permite organizar sus elementos hijos en filas o columnas.

Con Flexbox, puedes:

  • Alinear y distribuir fácilmente los elementos en horizontal o vertical.
  • Ajustar el tamaño de los elementos para que ocupen más o menos espacio según el tamaño de la pantalla.
  • Reorganizar el orden de los elementos sin cambiar el HTML.

Es ideal para crear layouts responsivos y manejar el espacio entre elementos de forma eficiente.
