Imágenes


            Las imágenes pueden mejorar el diseño y la apariencia de una página web.
                
            La etiqueta HTML <img> se utiliza para incrustar una imagen en una página web.

            Las imágenes no se insertan técnicamente en una página web; las imágenes se vinculan a las páginas web. La etiqueta <img> crea un 
            
            espacio reservado para la imagen referenciada.
            
            La etiqueta <img> es vacía, solo contiene atributos y no tiene una etiqueta de cierre.
            
            La etiqueta <img> tiene dos atributos obligatorios:
            
            <src> - Especifica la ruta de la imagen
            <alt> - Especifica un texto alternativo para la imagen
